HTML

​ “什么是HTML?”

HTML 是用来描述网页的一种语言。

  • HTML 指的是超文本标记语言: HyperText Markup Language
  • HTML 不是一种编程语言，而是一种标记语言
  • 标记语言是一套标记标签 (markup tag)
  • HTML 使用标记标签来描述网页
  • HTML 文档包含了HTML 标签及文本内容
  • HTML文档也叫做 web 页面

HTML标签

  • HTML 标签是由尖括号包围的关键词，比如 <html>
  • HTML 标签通常是成对出现的，比如 <b> 和 </b>
  • 标签对中的第一个标签是开始标签，第二个标签是结束标签
  • 开始和结束标签也被称为开放标签和闭合标签
  • 但是也有单个标签（称为自结束标签）：<标签>

HTML元素

​ HTML 文档由 HTML 元素定义。

​ html元素组成：开始标签+元素内容+结束标签（开始标签也称起始标签，结束标签也称闭合标签）

html元素部分语法

  • 某些html元素具有空内容

  • 空元素在开始标签中进行关闭（这两点简单地说，就是指一些标签（空元素）可以单独存在；如：<br>）

  • 大多数html元素可拥有属性

html空元素
  • 没有内容的 HTML 元素被称为空元素。空元素是在开始标签中关闭的。
  • <br> 就是没有关闭标签的空元素（<br> 标签定义换行）。
  • 在开始标签中添加斜杠，比如 <br />，是关闭空元素的正确方法，HTML、XHTML 和 XML 都接受这种方式。
  • 即使 <br> 在所有浏览器中都是有效的，但使用 <br /> 其实是更长远的保障。

html标签对大小写不敏感<P>等同于<p>；但是尽量用小写，因为万维网联盟（W3C）在 HTML 4 中推荐使用小写，而在xHTML 版本中强制使用小写。

html属性

  • 属性是html元素提供的附加信息

  • html元素可以设置属性

  • 属性在开始标签中描述

  • 属性总以名值对的形式出现，比如：name=”value”(可以单引号也可以双引号)

  • 属性与属性、属性与标签之间空一个间隔 ，如：**<标签 name=”value” name=’value’>**

html语义化标签

  • 在html中应该注重标签表示的语义，而不是在浏览器中显示的效果(这是css的工作)

    如h1标签表示为一级标题(重要性高)，而不是在于其在浏览器中的大小效果(css可以修改其大小)

  • 在页面中独占一行的元素称为块元素(block element)

  • 在页面中不会独占一行的元素称为行内元素/内联元素(inline element)

  • 一般块元素内能放行内元素(也能放块元素)，但是不会在行内元素内放块元素

    • p元素内不能放块元素
    • a元素内能放除了自身标签外的所有元素

  • 在浏览器解析网页时会自动对网页中不规范的内容进行修正

    • 比如：元素写在根元素的外部

      ​ p元素中嵌套了块元素

      ​ 根元素内出现了除head和body以外的子标签

      • 修正发生在主机内存中(通过F12开发者工具中element查看)，而不会修改源码

  • 结构化语义标签(content sectioning)

    • 标签 意义
      header 表示网页的头部 html5新加，较少使用
      main 表示网页的主体部分(一个页面中只会有一个main) 同上
      footer 表示网页的底部 同上
      nav 表示网页中的导航 同上
      aside 和主体相关的其他内容(侧边栏) 同上
      article 表示一个独立的文章 同上
      section 表示一个独立的区块，上边的标签都不能表示时使用section 同上
      div 没有语义，就用来表示一个区块，目前来讲div还是我们主要的布局元素 较多使用
      span 没有任何的语义，一般用于在网页中选中文字 较多使用

图片标签

  • 在html中，图像由<img>标签定义

    • img标签是自结束标签(也称空标签，意思是说，它只包含属性，并且没有闭合标签)
    • img标签属于嵌入标签
    属性 用法 作用
    src <img src="url" > 源属性src 指 “source”。源属性的值是图像的 URL 地址。
    alt <img src="url" alt="文本"> 当浏览器无法载入图像时，替换文本属性可告诉读者他们失去的信息。此时，浏览器将显示这个替代性的文本而不是图像。
    width <img src="url" width="intpx"> 宽，单位像素(px),若单独设置，则会按图片比例缩放
    height <img src="url" height="intpx"> 高，单位像素,若单独设置，则会按图片比例缩放

使用外链src引用的图片加载速度较慢，若需要与页面同步加载图片的话可以将图片转为base64编码，然后作为src的值

css简介

​ “什么是css？”

  • CSS 是一种描述 HTML 文档样式的语言

    （和html一样，也不属于编程语言，因为没有计算和逻辑判断能力）

  • CSS 指的是层叠样式表 (Cascading Style Sheets)

  • CSS 描述了如何在屏幕、纸张或其他媒体上显示 HTML 元素

    (包括针对不同设备和屏幕尺寸的设计和布局)

  • CSS 节省了大量工作。它可以同时控制多张网页的布局

  • 外部样式表存储在 CSS 文件中

css作用的方式

  1. 外部样式表（外部css）
  2. 内部样式表（内部css）
  3. 内联样式（行内css）
1.外部样式表

​ 内部样式直接在html文件内的head元素内使用style元素

​ style标签内的语法属于css，和html无关

3.内联样式

​ 用于为单个元素设置css样式。直接在需要设置样式的元素内使用style属性
